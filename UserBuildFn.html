FUNCTIONS





User-Defined functions:

A user-defined function is a block of code written by the user that performs a specific task and is executed only when it is called

Syntax:

def function name (parameters):
"""optional docstring"""
statements
return value

Example:

def add numbers (a, b):
sum = a + b
return sum
result = add numbers (10, 20)
print("Sum:", result)
#Output:
#Sum: 30
  • The function add numbers () is user-defined.
  • It takes two arguments (a and b), calculates their sum, and returns the result.
  • The function is executed only when it is called in the program.

Parts of a User-defined Function :

A Python function generally consists of the following five parts:
  1. Function Definition
  2. Functiom Body
  3. Function Call
  4. Function Parameters (Arguments)
  5. Return Statements

1) FUNCTION DEFINITION

This part tells Python that a function is being defined. It begins with the keyword def followed by the function name and parentheses ().

Example:

def greet ():
print ("Hello, Welcome to Python!")

2) FUNCTION BODY

The body contains the statements that perform the task. It can include calculations, loops, conditions, or other function calls.

Example:

def square (num):
result = num * num
return result

3) FUNCTION CALL

A function is executed only when it is called. To call a function, write its name followed by parentheses () and pass arguments if needed.

Example:

print (square (4))

4) FUNCTION PARAMETERS [Arguments]

Parameters are values passed to a function when it is called. They allow the function to work with different inputs.

Example:

def greet (name) :
print("Hello,", name)
#Call:
greet ("Anu")

5) RETURN STATEMENTS

The return statement is used to send the result from the function back to the part of the program that called it. If there is no return statement, the function returns None by default.

Example:

def multiply(a, b):
return a* b

Types of User-defined Functions

Python allows different types of user-defined functions depending on parameters and return values.

1) Function without Parameters and without Return Value

  • This type of function does not take any input and does not return any value.
  • It simply performs a task.

Example:

def welcome ():
print ("Welcome to Python Programming!")
welcome ()
#Output:
Welcome to Python Programming!

2) Funetion with Parameters but without Return Value

  • This function accepts input values but does not return any result.
  • It performs an operation and displays the result inside the function.

Example:

def display_name (name):
print ("Hello,", name)
display_name ("Meera")
#Output:
Hello, Meera

3) Funetion without Parameters but with Return Value

This function does not take any arguments but returnsa value to the caller.

Example:

def get_pi():
return 3.14159
print ("Value of pi:", get_pi())
#Output:
Value of pi: 3.14159


4) Function with Parameters and with Return Value



  • This is the most commonly used type.
  • It accepts arguments, performs a task, and returns a result.


Example:


def add(a, b):
return a + b
result = add(5, 7)
print("Sum =", result)
Output:
Sum = 12
