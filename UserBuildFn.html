User-Defined functions:

A user-defined function is a block of code written by the user that performs a specific task and is executed only when it is called

Syntax:

def function name (parameters):
"""optional docstring"""
statements
return value

Example:

def add numbers (a, b):
sum = a + b
return sum
result = add numbers (10, 20)
print("Sum:", result)
#Output:
#Sum: 30
